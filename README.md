# nuxt-demo

Nuxt.js Demo for my 2019 talks.

[See demo](https://nuxt-demo.surge.sh).

## Setup

```bash
yarn install
```

## Development

```
yarn dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Production

```
yarn build
yarn start
```

OR for static exporting:

```
yarn generate
```
